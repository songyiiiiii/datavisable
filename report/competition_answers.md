### 文字内容

采用 **PyVista 多层级等值面渲染** 管线，对 Nyx 128³ 密度场进行离屏体渲染。设计 8 层半透明等值面，密度阈值 [8.0–14.0] ln(ρ)。传递函数采用 6 段色彩映射：深海军蓝 (#0D2659) → 蓝绿 (#2E8CAD) → 翠绿 (#38B894) → 紫 (#8073B8) → 红 (#EB4747) → 亮金 (#FFD940)。不透明度从 0.06 (空洞) 递增至 0.68 (核心)。相机位置 (300, 280, 260)，焦点 (64, 64, 64)。每帧嵌入三个正交切片平面 (coolwarm, α=0.60) 及白色十字准线。100 帧批量渲染，单帧耗时约 2.0 秒。

从渲染结果观察到：（1）t=0–30，物质弥散，蓝色低密度主导；（2）t=30–50，红色高密度节点凝聚，纤维结构初现；（3）t=50–70，空洞膨胀，金色峰值出现；（4）t=70–99，宇宙网完全成型——深蓝空洞与红金节点两极分化，纤维骨架清晰。

### 文字内容

通过对 100 个时间步体渲染和统计数据的综合分析，归纳"三阶段演化模型"：

**第一阶段：线性增长期（t=0–30）**。初始密度高度均匀（偏度 0.716，峰度 1.63）。标准差 σ 从 0.432 缓慢升至 0.445，空洞占比从 0.31% 微增至 0.65%。

**第二阶段：非线性坍缩期（t=30–70）**。引力不稳定性触发加速聚集。σ 增至 0.48，IQR 扩大 16.7%，P99–P01 极差拓宽 15.1%。红色团簇增多，蓝色空洞指数膨胀（占比升至 1.8%）。

**第三阶段：准稳态期（t=70–99）**。演化减缓但未停。峰值密度突破 14.0，空洞占比达 2.99%（增长 9.7 倍）。纤维网络骨架完全形成。

**结构形成机制**：转移矩阵显示（1）仅 33% Node 体素保持节点状态——节点高度动态；（2）74.8% Void 保持空洞——空洞极其稳定；（3）Filament 中 48%→Sheet, 51%保留, 0.8%→Node——纤维是物质"高速公路"。

**可视化技术价值**：体渲染将 3D 密度场映射为连续色彩，多层半透明叠加使内部结构可见；相图揭示"空洞→纤维→团簇→核心"四象限统计基础；转移矩阵量化不可逆结构转变；时间轮展示高维统计特征的演化轨迹。

| 图3.5 | 直方图刷选联动粒子云 | `src/three/ParticlePanel.jsx` + `src/App.jsx`（联动逻辑） |

### 文字内容

对每时间步 2,097,152 个密度值进行全局统计分析，计算均值、标准差、偏度、峰度、IQR、百分位数、空洞占比和峰值占比。构建 128 区间密度直方图（7.5–14.5），计算逐步时间导数 ∂H/∂t。

**核心发现**：

分布展宽：均值 9.485→9.535（+0.53%），标准差 0.432→0.498（+15.4%）。概率密度函数呈经典"胖尾化"——质量从中密度区间（9.0–10.5）不可逆地流向两端。

两极分化：空洞占比 0.31%→2.99%（+9.7 倍），峰值占比 0.024%→0.13%（+5.5 倍）。偏度全程正值（0.716→0.718），确认右偏贯穿演化全程。峰度 1.63→1.45，表明极端离群值相对贡献降低，对应高密度尾部"多中心化"的物理过程。

∂H/∂t 分析显示 t=35–45 区间出现最显著的直方图形变加速（"引力坍缩触发期"），t=70 之后趋近稳态。

**刷选验证**：框选 ln ρ ∈ [12.5, 14.5]（占比 0.08%），3D 视图过滤显示仅 ~600 体素，高度聚集为 5–8 个致密核心——证实"直方图高密度尾部对应宇宙网节点"的物理对应。

### 文字内容

构建基于 React + ECharts + Three.js 的联动式交互可视分析系统。核心交互逻辑：用户在左侧直方图中使用 ECharts Brush 拖拽选择密度区间 → 系统实时计算占比和空间属性 → 三维粒子云同步过滤 → 分析面板自动生成统计描述。

**刷选实验结果**：

| 区间 | 占比 | 空间形态 |
|------|------|----------|
| [7.5, 8.5] | 2.99% | 大范围弥散 → 空洞 |
| [9.0, 10.0] | 58.2% | 均匀填充 → 薄片/背景 |
| [10.5, 11.5] | 8.7% | 沿纤维排列 → 宇宙网丝状结构 |
| [12.0, 13.0] | 0.13% | 数个致密团块 → 星系团外围 |
| [13.5, 14.5] | 0.008% | 3–5 个核心 → 星系团中心 |

**双向验证闭环**：（1）ln ρ > 13.5 选取后粒子云集中分布在 3–5 个致密核心，与体渲染中红色高亮团簇节点完全重合；（2）ln ρ < 8.5 选取后粒子呈现弥散均匀分布，对应体渲染蓝色空洞区域。系统实现了"统计选空间、空间验证统计"的闭环工作流：直方图的展宽是物质从均匀分布向团块化演变的统计投影，而刷选联动则使这种统计-空间对应关系变得直观可验证。

**交互指标**：刷选响应延迟 < 100ms (ECharts Brush → 粒子云更新)，支持 5 区间快速切换对比。面板宽度和高度均可拖拽调节（250–550px），进度条驱动 100 帧时间轴。
