# 任务二：宇宙密度演化特征分析 — 技术报告

> **共享数据描述**：数据集格式、处理平台、处理原则、分类体系等通用信息参见 [`data_description.md`](data_description.md)。本文档仅包含任务二特定的技术内容。

## 一、问题分析

### 1.1 核心目标

基于任务一的体渲染可视化结果与任务三的统计直方图数据，从定性（视觉特征）和定量（统计指标）两个维度，系统归纳宇宙密度场的演化规律。需要回答三个核心问题：

1. **宇宙结构如何形成？** 从均匀初始态到团块化终态的演化路径是什么？
2. **星系际介质如何演化？** 低密度的空洞和中等密度的纤维网在演化中扮演什么角色？
3. **可视化技术在宇宙学数据分析中的应用价值是什么？** 体渲染与统计图表如何互补？

### 1.2 技术挑战

- **多维统计指标的同步追踪**：100 个时间步中，需要同时追踪均值、中位数、标准差、IQR、偏度、峰度、P1、P99 共 8 个指标，并识别它们之间的关联。
- **视觉特征与统计指标的对应**：需要建立体渲染图像中的视觉变化（蓝色扩张、红色收缩）与统计数值之间的定量对应关系。
- **非线性演化阶段的识别**：引力坍缩是一个非线性过程，需要在连续的时间序列中识别出演化速率变化的转折点。

## 二、方法论

### 2.1 统计指标体系

为全面刻画密度分布的演化，构建以下统计指标：

| 指标 | 定义 | 物理含义 |
|------|------|---------|
| μ（均值） | $\frac{1}{N}\sum \rho_i$ | 全局平均密度，下降反映空洞体积占比增大 |
| σ（标准差） | $\sqrt{\frac{1}{N}\sum(\rho_i-\mu)^2}$ | 密度离散度，上升=团块化加剧 |
| IQR | P75 − P25 | 稳健离散度，排除极端值影响 |
| γ₁（偏度） | $\frac{\mu_3}{\sigma^3}$ | 分布不对称性，正值=右尾发育 |
| γ₂（超额峰度） | $\frac{\mu_4}{\sigma^4} - 3$ | 尾重程度，>0=厚尾 |
| P1, P99 | 第 1、99 百分位 | 极端值追踪 |
| P99−P01 跨度 | — | 两极分化幅度 |

### 2.2 五图协同分析框架

设计五张互补图表，形成"全景→聚焦→对比→关联→量化"的分析链：

| 图序 | 类型 | 分析功能 |
|------|------|---------|
| Fig 2a | 四格统计面板 | 全局演化概览：中心趋势、离散度、形态、极值 |
| Fig 2b | 小提琴序列 | 分布形态追踪：直观展示形状演变 |
| Fig 2c | 直方图叠加 | 初始-最终直接对比：t=0 vs t=99 |
| Fig 2d | 六边形联合分布 | 单一体素命运追踪：哪些体素密度增大了？ |
| Fig 2e | 六指标量化面板 | 团块化多维度量化：每个指标的趋势线+百分比变化 |

### 2.3 关键分析技术

**小提琴图**：每 5 步取一个时间步，随机采样 50,000 体素，使用 matplotlib 的 `violinplot` 绘制。体素均值趋势线（红色虚线）叠加在小提琴图上，直观展示分布中心的下移。

**六边形联合分布**：从 2,097,152 体素中随机采样 20,000 对 (t=0, t=99) 密度值，使用 `hexbin` 以对数色彩映射绘制二维直方图。1:1 对角线（灰色虚线）标记"无变化"参考线；对角线上方的体素为"密度增大"（坍缩），下方为"密度减小"（疏散）。

## 四、结果与分析

### 4.1 不可逆团块化（Irreversible Clumpification）

标准差 σ 和 IQR 的单调递增（分别 +15.4% 和 +16.7%）是最直接的团块化证据。线性拟合显示两个指标的增长近似线性（R² > 0.98），表明 100 个时间步中团块化过程尚未达到饱和。标准差从 0.432 增长到 0.498 意味着密度分布的"宽度"增加了约 1/6——在 2,097,152 体素的尺度上，这对应着数百万体素从中间密度迁移到高密度或低密度区域。

### 4.2 两极分化（Bipolarization）

最小值和最大值的反向运动（最小值下降 0.231，最大值上升 0.606）表明物质同时在向两个极端流动。最大值的变化幅度（+0.606）约为最小值变化幅度（−0.231）的 2.6 倍，说明引力坍缩（物质聚集）的效应强于空洞扩张（物质疏散）的效应。这与冷暗物质（ΛCDM）模型的预期一致：引力驱动的结构形成是一个"富人更富"的正反馈过程。

### 4.3 分布形态的非对称化

偏度从 0.716 增长到 0.718，虽然绝对变化不大，但结合小提琴图（Fig 2b）可以清楚看到：早期分布近似对称（小提琴形态在 t=0 时呈纺锤形），而晚期分布明显右偏（t=99 时小提琴上部显著拉长）。偏度正值表明高密度尾在发育——这正是引力坍缩将物质从中间密度"泵送"到高密度的直接统计证据。

### 4.4 可视化技术的应用价值

1. **体渲染（任务一）提供空间拓扑信息**：三维渲染直接展示了空洞、纤维、团簇的空间排列关系，这是纯统计图表无法传达的。
2. **统计图表（任务二）提供定量约束**：小提琴图和直方图叠加将定性视觉印象转化为精确的数值比较。
3. **联合分布（Fig 2d）揭示个体命运**：六边形图追踪了单一体素的密度演化轨迹，使得从"群体统计"到"个体运动"的推理成为可能。
4. **互补性**：体渲染回答"结构在哪里？"，统计图表回答"结构变化了多少？"，联合分析回答"哪些区域变化了？"——三者共同构成了完整的宇宙密度演化图景。

## 五、结论

Nyx 模拟的 100 个时间步揭示了暗物质引力驱动下宇宙密度场从均匀到结构的完整演化路径。统计指标（σ +15.4%, IQR +16.7%, P99−P01 +15.1%）一致表明团块化是单调且不可逆的过程；偏度和峰度的变化揭示了分布形态从近对称到右偏非对称的转变。体渲染与统计图表的协同使用证明了可视化技术在宇宙学数据分析中的核心价值：前者提供空间直觉，后者提供定量约束。
