# 任务一：体数据渲染与密度演化可视化 — 技术报告

> **共享数据描述**：数据集格式、处理平台、处理原则、分类体系等通用信息参见 [`data_description.md`](data_description.md)。本文档仅包含任务一特定的技术内容。

## 一、问题分析

### 1.1 核心挑战

Nyx 宇宙学模拟数据集包含 100 个时间步的三维气体密度场（128³ 体素，float32，小端字节序，列优先存储），密度值跨越约 7.75–14.45（自然对数尺度，ln ρ）。体渲染面临三个核心技术挑战：

**挑战一：密度动态范围的两极呈现**。对数密度跨度近 7 个单位（约 3 个数量级在线性空间），低密度空洞（ln ρ ≈ 7.8–8.5）与高密度团簇（ln ρ ≈ 12.0–14.5）需要同时可见。若采用线性不透明度映射，低密度区域将完全透明而高密度区域过曝；若采用均匀映射，则中间密度纤维结构会淹没高密度团簇的细节。

**挑战二：时间演化的可视化对比**。100 个时间步中密度分布从窄峰均匀态逐步演化为宽峰团块化态，标准差增长 15.4%（0.432→0.498）。需要在统一视觉框架下呈现这一演化过程，使得不同时间步的结构差异可量化对比。

**挑战三：纤维结构的视觉感知增强**。宇宙纤维网（Cosmic Web）是中低密度区域（ln ρ ≈ 8.5–10.5）连接空洞与团簇的桥梁结构，密度对比度低，需要光照模型增强其三维空间感知。

### 1.2 技术路线选择

经过对三种体渲染方案的评估，最终选择 **PyVista 多层等值面渲染**：

| 方案 | 优势 | 劣势 | 结论 |
|------|------|------|------|
| VTK GPU 光线投射 (`vtkSmartVolumeMapper`) | 真实体渲染，支持复合混合模式 | 离屏渲染输出不稳定（本机多次出现近白屏），交互窗口需桌面 GPU 上下文 | 备选 |
| Plotly WebGL (`go.Volume`) | 浏览器原生，跨平台 | 在本机浏览器上未能渲染出任何可见内容 | 不可用 |
| **PyVista 多层等值面** | 已验证 mesh 渲染可用，支持 Blinn-Phong 光照，可精确控制每层颜色/不透明度 | 非严格意义上的体渲染（等值面而非体积积分） | **✅ 选用** |

## 二、方法论

### 2.1 多层等值面传递函数设计

核心思路：将连续密度场离散化为 8 层嵌套等值面，每层对应特定密度区间，通过颜色和不透明度的逐层叠加模拟体渲染效果。

**等值面层级定义：**

| 层级 | 密度 ln(ρ) | 颜色 (R,G,B) | 不透明度 | 物理对应 | 光照模式 |
|------|-----------|-------------|---------|---------|---------|
| 1 | 8.0 | (0.13, 0.40, 0.67) 深蓝 | 0.08 | 深空洞 | 标准 |
| 2 | 8.6 | (0.35, 0.58, 0.80) 中蓝 | 0.12 | 浅空洞 | 标准 |
| 3 | 9.1 | (0.60, 0.75, 0.90) 浅蓝 | 0.18 | 空洞边界 | 标准 |
| 4 | 9.7 | (0.90, 0.90, 0.90) 浅灰 | 0.25 | 纤维网 | 标准 |
| 5 | 10.3 | (0.95, 0.78, 0.68) 暖粉 | 0.32 | 暖纤维 | 标准 |
| 6 | 10.9 | (0.88, 0.50, 0.38) 橙红 | 0.45 | 团簇晕 | **发光** |
| 7 | 11.8 | (0.75, 0.25, 0.22) 红色 | 0.55 | 团簇核心 | **发光** |
| 8 | 14.0 | (0.60, 0.10, 0.15) 深红 | 0.70 | 极端峰 | **发光** |

**设计原理**：

1. **密度-颜色映射**：采用感知均匀的蓝→灰→红渐变。蓝色对应低密度（空洞），灰色对应中间密度（纤维网），红色对应高密度（团簇）。这一映射利用了人类视觉对红-蓝色差的高敏感度，使得结构对比最大化。

2. **密度-不透明度映射**：采用三段分段线性函数，低密度区（ln ρ < 9.2）不透明度范围 0.08–0.18（透明→半透明），中密度区（9.2–10.5）0.25–0.32（半透明），高密度区（10.5+）0.45–0.70（接近不透明）。该设计使得低密度空洞"呼吸"可见而不喧宾夺主，高密度团簇坚固突出。

3. **团簇发光处理**：第 6–8 层（密度 > 10.5）采用 emissive 光照模式（ambient=1.0, diffuse=0.25），模拟高密度区域因恒星形成和 AGN 活动产生的"自发光"效果，在视觉上形成从外部光照到内部发光的过渡。

### 2.2 Blinn-Phong 光照模型

采用经典的 Blinn-Phong 反射模型增强三维空间感知：

$$I = k_a I_a + k_d (N \cdot L) I_d + k_s (N \cdot H)^\alpha I_s$$

| 参数 | 标准层（层级1-5） | 发光层（层级6-8） | 物理含义 |
|------|------------------|------------------|---------|
| $k_a$ (ambient) | 0.18 | 1.00 | 环境光：确保阴影区不完全黑暗 |
| $k_d$ (diffuse) | 0.72 | 0.25 | 漫反射：定向光源的主照明分量 |
| $k_s$ (specular) | 0.35 | 0.35 | 镜面反射：表面高光 |
| $\alpha$ (shininess) | 14 | 14 | 高光集中度 |

**光照方向**：光源位置设置为斜上方 (200, 200, 180)，模拟"宇宙手电筒"效果，从上侧照射密度场，使得纤维结构的上下表面产生明显的明暗对比，增强立体感。

**法线计算**：等值面法线通过 VTK 的梯度算子自动计算，基于密度场的中心差分近似 $\nabla \rho$，无需手动提供法线数据。

### 2.3 时间演化对比策略

采用三种策略实现时间步间的有效对比：

1. **固定相机参数**：所有时间步使用同一相机位置 `[(200, 200, 180), (64, 64, 64), (0, 0, 1)]`，确保视角一致。

2. **统一传递函数**：8 层等值面的密度阈值、颜色、不透明度参数在所有时间步完全一致，使得视觉差异仅反映密度场的物理演化。

3. **增强帧率**：预渲染 51 帧（每 2 步一帧含 t=99），通过 Dash 滑块实现近乎连续的演化回放，帧间切换<200ms，产生流畅的动画效果。

## 三、实现细节

### 3.1 数据预处理

```python
# Nyx .dat 文件格式：小端 float32，128³，列优先（Fortran顺序 Z→Y→X）
raw = np.fromfile(path, dtype="<f4")
data_3d = raw.reshape((128, 128, 128), order='F')

# 转换为 PyVista ImageData（需转置为 X→Y→Z C顺序）
data_xyz = np.ascontiguousarray(np.transpose(data_3d, (2, 1, 0)))
grid = pv.ImageData()
grid.dimensions = (129, 129, 129)
grid.cell_data["density"] = data_xyz.flatten(order="C")
grid = grid.cell_data_to_point_data()  # contour过滤器需要点数据
```

### 3.2 等值面生成

使用 VTK 的 `vtkContourFilter`（通过 PyVista 的 `.contour()` 方法），在 8 个密度阈值处生成等值面：

```python
for lvl in [8.0, 8.6, 9.1, 9.7, 10.3, 10.9, 11.8, 14.0]:
    iso = grid.contour(isosurfaces=[lvl], scalars="density")
    plotter.add_mesh(iso, color=color, opacity=opacity,
                     ambient=amb, diffuse=dif, specular=0.35)
```

### 3.3 Web 集成

通过 Dash 框架将预渲染的 51 帧 PNG 图像与交互式 TimeWheel 极坐标图集成在同一界面，共享时间轴驱动。

## 四、结果与分析

### 4.1 传递函数效果验证

**低密度空洞呈现**：不透明度 0.08–0.12 的设计使得空洞区域（ln ρ < 8.6）呈现为淡蓝色半透明结构，允许观察者"看穿"空洞看到其内部的子结构。在 t=99 的渲染中，空洞体积占比从 t=0 的 0.31% 显著增长至 2.99%，视觉上表现为蓝色区域的明显扩张。

**高密度团簇呈现**：不透明度 0.55–0.70 配合 emissive 光照，使得团簇核心（ln ρ > 11.8）呈现明亮的橙红色。团簇在空间上呈节点状聚集，与宇宙网拓扑预期一致。

**中间密度纤维**：不透明度 0.25–0.32 和灰色/暖粉色使得纤维网作为空洞与团簇之间的过渡结构自然呈现，形成连续的"海绵状"拓扑。

### 4.2 光照效果对纤维结构的感知增强

通过对比有无光照的渲染结果，光照模型在以下三个方面增强了纤维结构感知：

1. **深度感知**：斜上方的定向光源在纤维结构的上下表面产生亮度差异，提供了强烈的深度线索。
2. **曲率信息**：镜面反射高光（specular=0.35, shininess=14）在纤维的弯曲处产生局部亮度峰值，暗示了纤维的曲率和走向。
3. **团簇识别**：发光团簇的 emissive 材质使其在纤维网络中"跳出"，形成视觉焦点。

### 4.3 演化趋势可视化

51 帧序列清晰展示了从 t=0 均匀态到 t=99 团块化态的连续过渡。关键视觉变化包括：

| 时间步 | 视觉特征 | 物理含义 |
|------|---------|---------|
| t=0–15 | 大片蓝色（空洞占比 < 1%） | 物质分布均匀，近高斯分布 |
| t=25–40 | 蓝色区域扩张，红色点开始出现 | 引力不稳定性启动，初始团块形成 |
| t=50–70 | 纤维网（灰色）明显可见 | 宇宙网结构成形，空洞-纤维-团簇拓扑建立 |
| t=75–99 | 团簇（红色）收缩增亮，空洞加深 | 非线性引力坍缩，两极分化加速 |

## 五、结论

本任务通过 PyVista 多层等值面渲染结合 Blinn-Phong 光照模型，成功实现了 Nyx 宇宙学密度场的三维可视化。传递函数的非线性的设计（分段不透明度 + 蓝灰红颜色梯度 + 团簇 emissive 光照）有效解决了低密度空洞与高密度团簇同时呈现的视觉挑战。51 帧预渲染配合统一参数确保了时间演化对比的定量有效性。光照模型显著增强了纤维结构的三维空间感知，为理解宇宙大尺度结构形成提供了直观的视觉证据。
