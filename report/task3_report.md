# 任务三：密度对数直方图时序追踪 — 技术报告

> **共享数据描述**：数据集格式、处理平台、处理原则、分类体系等通用信息参见 [`data_description.md`](data_description.md)。本文档仅包含任务三特定的技术内容。

## 一、问题分析

### 1.1 核心目标

在 100 个演化时间步中，自主统计每个步长的气体密度数据，构建密度对数直方图，量化追踪全域密度分布随时间的演化偏移规律。

### 1.2 技术挑战

- **高维数据的降维表示**：100 个时间步 × 128 个密度区间的数据矩阵（100×128），需要设计既能展示全貌又不丢失细节的可视化方案。
- **分布偏移的量化**：不能仅依赖"看图说话"，需要引入可计算的偏移度量指标。
- **极端值区域的追踪**：空洞（< P1）和峰值（> P99）只占总体的 2%，但对物理理解至关重要。
- **变化速率的可视化**：不仅要展示"变化了什么"，还要展示"变化有多快"。

## 二、方法论

### 2.1 五图分析框架

| 图序 | 类型 | 分析功能 | 技术 |
|------|------|---------|------|
| Fig 3a | 演化热力图 | 全局概览：100步×128区间的密度分布全貌 | `pcolormesh` + 对数色彩映射 |
| Fig 3b | 峰位/宽度追踪 | 分布中心与展宽的定量演化 | 模式+ FWHM 计算 |
| Fig 3c | 空洞/峰值占比 | 两极极端区域的体积分数演化 | 绝对密度阈值（ln<8.5 / >12.0） |

| Fig 3e | 山脊图 | 关键时间步的直方图叠加 | 11 条偏移填充曲线 |

### 2.2 关键算法

**直方图构建**：
```python
hist, edges = np.histogram(data_3d.ravel(), bins=128, range=(7.5, 15.0))
# 128 个等宽区间，覆盖 7.5-15.0 的完整密度范围
```

**FWHM（半高全宽）计算**：
```python
half_max = hist[t].max() / 2
above = np.where(hist[t] >= half_max)[0]
fwhm[t] = bin_centers[above[-1]] - bin_centers[above[0]]
# FWHM 是衡量分布宽度的稳健指标，不受极端离群值影响
```

**时间导数（分布变化率）**：
```python
deriv[t] = (hist[t+1] - hist[t-1]) / 2  # 中心差分
# 正值（红色）= 该密度区间的体素增加
# 负值（蓝色）= 该密度区间的体素减少
```

**绝对密度阈值**（区别于百分位数阈值）：
```python
void_frac[t] = (data[t] < 8.5).sum() / N_VOXELS  # ln(rho) < 8.5
peak_frac[t] = (data[t] > 12.0).sum() / N_VOXELS  # ln(rho) > 12.0
# 使用绝对阈值而非百分位数（P5/P95始终是5%，无法反映真实变化）
```

## 三、实现细节

### 3.1 演化热力图

- **色彩映射**：`RdBu_r`（红-白-蓝反转），蓝色 = 低计数（早期窄峰），红色 = 高计数（晚期宽峰的中心区域）
- **对数归一化**：`log10(hist + 1)` 增强低计数区域的可见性
- **标注**：5 条虚线标记代表性时间步（t=0/25/50/75/99）

### 3.2 空洞/峰值占比

选择 ln(ρ) < 8.5 作为空洞阈值，ln(ρ) > 12.0 作为峰值阈值的依据：
- t=0 时，ln(ρ) < 8.5 的体素占比仅 0.31%，对应分布的最左尾
- t=99 时，ln(ρ) < 8.5 的体素占比增长至 2.99%，接近 10 倍增长
- ln(ρ) > 12.0 在 t=0 时几乎为零（0.02%），t=99 时增长至 0.03%

## 四、结果与分析

### 4.1 分布演化的三阶段模型

基于演化热力图（Fig 3a）和时间导数（Fig 3d），识别出分布演化的三个特征阶段：

| 阶段 | 时间范围 | 热力图特征 | 导数特征 | 物理过程 |
|------|---------|----------|---------|---------|
| I：线性增长 | t=0–30 | 窄峰、高中心 | 导数均匀分布 | 初始引力不稳定性 |
| II：加速演化 | t=30–70 | 峰宽化、尾发育 | 两端正、中间负 | 非线性坍缩启动 |
| III：渐近趋稳 | t=70–99 | 宽峰、长尾 | 导数减速 | 准平衡态逼近 |

### 4.2 分布峰位偏移的物理含义

分布模式（Mode）从 t=0 的 9.404 下降到 t=99 的 9.229（-0.176），这意味着"最典型"的密度值在向低密度方向移动。这看似矛盾——引力坍缩应该使物质更致密——但实际反映了宇宙空洞的体积膨胀效应：随着空洞区域扩大，更多体素进入低密度区间，"典型密度"被拉低。

### 4.3 时间导数的双向流特征

Fig 3d 是所有图中信息密度最高的。它揭示了一个"双向流"模式：
- **低密度端（ln ρ < 8.5）**：持续正值（蓝色→红色过渡），表示空洞体素在不断增加
- **中间密度（9.0–10.5）**：持续负值（蓝色），表示纤维区的体素在流失
- **高密度端（ln ρ > 12.0）**：持续正值（红色），表示团簇核心在积累物质

这一模式是"质量从中间向两端转移"的直接可视化证据——引力两极分化的统计签名。

### 4.4 山脊图的视觉修辞

山脊图（Fig 3e）采用"数据山脉"的视觉隐喻：每条曲线是一个时间步的密度分布，曲线的"高度" = 体素计数，"位置" = 密度值。早期时间步（蓝色）形成"尖峰"——分布集中、形态对称；晚期时间步（红色）形成"矮丘"——分布扩散、右偏拉长。视觉上的"山脉侵蚀"对应物理上的"团块化扩散"。

## 五、结论

密度对数直方图的 100 步时序追踪揭示了宇宙密度场从均匀窄峰分布向宽尾双极分布的完整演化过程。分布峰位下降 0.176、FWHM 增长 14.3%、空洞体积占比增长约 10 倍——这些定量指标共同描绘了暗物质引力驱动下物质从中间密度向两极流动的系统性过程。时间导数热力图进一步揭示了这一过程的非均匀速率，为理解引力坍缩的非线性动力学提供了统计视角。
